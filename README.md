## 📌 Sobre o Projeto

O **Muniz Imóveis Admin** é uma aplicação Front-end robusta desenvolvida para simplificar a rotina de corretores e administradores de imóveis. Ele atua como o "cérebro" da imobiliária, permitindo o gerenciamento completo de propriedades, contratos de aluguel, inquilinos e configurações do sistema em uma interface limpa, intuitiva e totalmente responsiva.

Este repositório contém exclusivamente o **Módulo Administrativo**. A arquitetura foi pensada em microserviços de front-end, preparando o terreno para a futura integração com a "Vitrine de Imóveis" (área do cliente final).

## ✨ Principais Funcionalidades

- **Dashboard Interativo:** Gráficos dinâmicos de faturamento e estatísticas em tempo real.
- **Gestão de Imóveis e Contratos:** Tabelas completas com paginação, filtros, ordenação e scroll horizontal adaptativo para mobile.
- **Autenticação e Segurança:** Rotas protegidas e controle de acesso integrado ao backend.
- **Design Responsivo & UX Premium:** Layout Mobile-First, garantindo fluidez em qualquer tamanho de tela sem quebras de layout.
- **Dark/Light Mode:** Suporte nativo a temas visuais com persistência de preferência do usuário.

## 🛠️ Tecnologias Utilizadas

A stack foi cuidadosamente escolhida para garantir performance, facilidade de manutenção e ferramentas de estado avançadas:

- **[React](https://reactjs.org/) + [Vite](https://vitejs.dev/)**: Construção da interface e tooling super rápido.
- **[React Router DOM](https://reactrouter.com/)**: Gerenciamento de rotas (Single Page Application).
- **[React Query (TanStack Query)](https://tanstack.com/query/latest)**: Sincronização, cache e gerenciamento do estado remoto.
- **[Styled Components](https://styled-components.com/)**: Estilização baseada em componentes (CSS-in-JS).
- **[Supabase](https://supabase.com/)**: Backend-as-a-Service (BaaS) para banco de dados PostgreSQL e Autenticação.
- **[Recharts](https://recharts.org/)**: Renderização de gráficos responsivos.
- **[React Hot Toast](https://react-hot-toast.com/)**: Feedback visual elegante para o usuário (notificações).
